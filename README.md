# Advanced Task Management System (ATMS)
The Advanced Task Management System (ATMS) is a real-time, collaborative task management tool built with React, Redux Toolkit, Firebase, and Framer Motion. Designed to facilitate team collaboration, this application allows users to create, assign, and track task progress in a professional, interactive, and engaging environment.

# Tech Stack
React, Redux Toolkit, Firebase, Chart.js, framer-motion

# Project Highlights
1. User authentication and authorization
   - firebase authentication that handles secure user sign-ups and logins
   - role-based access control for admins and users
2. Real-Time Task Updates
   - real-time updates on tasks using firebase
3. Task Management
   - users can create, assign, and track tasks with various statuses like         "To Do", "In Progress" and "Completed".
   - Tasks are displayed with dynamic data updates using Firebase and Redux
4. Interactive UI
   - framer motion animations bring a responsive and engaging user                experience to the app's modals, charts and notifications
5. Data visualization
   - Task progress charts built with Chart.js and Framer Motion provide           intiuitive insights into team producivity
